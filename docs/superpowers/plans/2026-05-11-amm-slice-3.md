## Math

**Add liquidity (Uniswap V2 "lesser side" algorithm):**
- Given user inputs `rpow_in`, `usdc_in`, pool `R_rpow`, `R_usdc`, `total_lp`:
- Compute `lp_minted = min(rpow_in √ó total_lp / R_rpow, usdc_in √ó total_lp / R_usdc)` (floor division).
- Then `rpow_used = lp_minted √ó R_rpow / total_lp` (floor), `usdc_used = lp_minted √ó R_usdc / total_lp` (floor).
- `rpow_refunded = rpow_in - rpow_used`, `usdc_refunded = usdc_in - usdc_used`. Refunds stay in user's balance (RPOW: leftover tokens NOT touched; USDC: `usdc_in - usdc_used` not debited from user).
- If `lp_minted < min_lp_out`: throw SLIPPAGE_EXCEEDED.
- Pool: `R_rpow += rpow_used`, `R_usdc += usdc_used`, `total_lp += lp_minted`.
- User: deduct `usdc_used` from `users.usdc_base_units`. Burn `rpow_used` from user via `burnFromUser` (which mints change if pick overshoots ‚Äî that's compatible with the refund concept).
- USDC cap check on `usdc_used` (the cap is on total USDC in system; adding LP moves USDC from user balance to pool reserve ‚Äî total unchanged, no check needed). Wait, that's correct ‚Äî no cap check on `lp/add`.

**Remove liquidity:**
- Given user's `lp_balance ‚â• lp_burn`, pool reserves, `total_lp`:
- `rpow_out = lp_burn √ó R_rpow / total_lp` (floor), `usdc_out = lp_burn √ó R_usdc / total_lp` (floor).
- If `rpow_out < min_rpow_out` or `usdc_out < min_usdc_out`: SLIPPAGE_EXCEEDED.
- Pool: `R_rpow -= rpow_out`, `R_usdc -= usdc_out`, `total_lp -= lp_burn`.
- User: `lp_balance -= lp_burn`. Mint RPOW token of value=`rpow_out` to user (cap-checked supply). Credit `users.usdc_base_units += usdc_out`.

**Helper bigint min:** define inline `function bmin(a, b) { return a < b ? a : b }`.

## LpEventPayload

```ts
export interface LpEventPayload {
  id: string;
  account_email_hash: string;
  type: 'ADD' | 'REMOVE';
  rpow_delta_base_units: bigint;       // signed: positive = caller-side gain
  usdc_delta_base_units: bigint;
  lp_delta_base_units: bigint;          // signed
  pool_rpow_after: bigint;
  pool_usdc_after: bigint;
  total_lp_after: bigint;
  created_at: string;
}
```

ADD: rpow_delta = -rpow_used, usdc_delta = -usdc_used, lp_delta = +lp_minted.
REMOVE: rpow_delta = +rpow_out, usdc_delta = +usdc_out, lp_delta = -lp_burn.
